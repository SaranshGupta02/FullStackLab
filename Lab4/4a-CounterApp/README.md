# Counter App (Assignment 4.a)

A React application that implements a button click counter with an input box to set the initial counter value.

## Features

- Input box to set initial counter value
- Button that increments counter by 1 on each click
- Reset button to reset counter to 0
- Modern, responsive UI with hover effects

## How to Run

1. Navigate to the `4a-CounterApp` directory
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Usage

1. Enter a number in the "Set Initial Value" input field
2. Click "Set Initial Value" to set the counter to that number
3. Click "Click Me! (+1)" to increment the counter
4. Click "Reset" to reset the counter to 0

## Technologies Used

- React 18.2.0
- JSX
- CSS3
- React Hooks (useState)
