# Lab 7 - ToDo List with useReducer

This lab demonstrates how to build a ToDo List application using React's `useReducer` hook for state management.

## Project

### ToDo List Application
- Location: `ToDoList-useReducer/`
- State Management: `useReducer` hook
- Features: Add, Edit, Complete, and Delete tasks
- Display: Tasks shown in a structured table format

## Requirements Fulfilled

✅ Text box with Add button for adding tasks  
✅ Tasks displayed in a table  
✅ First column: Serial Number (S.No)  
✅ Second column: Task description  
✅ Third column: Edit button  
✅ Fourth column: Complete and Delete buttons  
✅ Uses `useReducer` hook for state management  

## Running the Project

```bash
cd ToDoList-useReducer
npm install
npm start
```

The application will run on `http://localhost:3000`

## Key Features

- **Add Tasks**: Input field with Add button to create new tasks
- **Edit Tasks**: Inline editing with Save/Cancel options
- **Complete Tasks**: Toggle completion status with Complete/Undo button
- **Delete Tasks**: Remove tasks permanently
- **Serial Numbers**: Automatic numbering starting from 1
- **Table Layout**: Clean, organized table display
- **Responsive Design**: Works on desktop and mobile devices
