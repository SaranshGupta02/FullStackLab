# Item List App (Assignment 4.b)

A React application containing an input box and two buttons: "Add Item" and "Show Items". Items are stored in a list and displayed in an unordered list.

## Features

- Input box to enter items
- "Add Item" button to add items to the list
- "Show Items" button to toggle display of all items
- Items displayed in an unordered list
- Clear All button to remove all items
- Item count display
- Enter key support for adding items
- Modern, responsive UI

## How to Run

1. Navigate to the `4b-ItemListApp` directory
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Usage

1. Type an item in the input field
2. Click "Add Item" or press Enter to add the item to the list
3. Click "Show Items" to display all stored items in an unordered list
4. Click "Hide Items" to hide the list
5. Click "Clear All" to remove all items from the list

## Technologies Used

- React 18.2.0
- JSX
- CSS3
- React Hooks (useState)
